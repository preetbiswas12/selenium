Create a Google Developer Profile

A Google Developer Profile is mandatory for registration. If you already have a profile, simply submit your profile link in the registration form.

If you do not have one, follow these steps:

  1. Visit https://developers.google.com/
  2. Sign in or sign up using the same email address you will use for hackathon registration.
  3. Complete your developer profile setup.
  4. Make sure your profile visibility is set to Public.
  5. Copy your Google Developer Profile link and submit it in the registration form.
